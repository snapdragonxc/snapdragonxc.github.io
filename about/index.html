About Software Engineer

December 22, 2017

I am a former Mechanical Engineer now working as a Software Engineer. I began my software career in 2000, when I wrote the mathematical code for my PhD thesis in C language (completed 2004). This was on the application of Artificial Neural Networks to engineering problems. At the end of four years of study I was hooked on software and in 2007 I began working as a Software Engineer at Serco Defence at HMAS Watsons Bay, a Navy training base. Here I learnt about object-orientated code, networking, and software engineering, coding first in C++ and then in C#. In recent years, I have moved on to Javascript where I write code on both the Front-End and Back-End. I specialise in ReactJS/NodeJS single-page applications (SPA). In the past I have built SPA using AngularJS.

Some of the work I have done:

Ecommerce

http://github.com/snapdragonxc/ecommerce

This is ReactJS/Redux/MongoDb/ExpressJS template that I have been developing since July 2017. It is designed for use in small business. The code is standard React/Redux using Flow for type-checking and Mocha/Chai/Supertest for integration testing. The styling was done with SASS following the BEM methodology. Payment integration is with Paypal. It includes a basic dashboard for managing stock.

You can see a working example, “Charm Accessories” at:

www.charmaccs.com

Charm Accessories is hosted on AWS EC2 with Continuous Integration/Deployment using a Jenkins EC2 server.

Front-End-Fwk

http://github.com/snapdragonxc/front-end-fwk

Front-End-Fwk is a javascript application that I have been developing since 2016. It is based on ExpressJS, only it is designed for the front end and not the back. You can see it in action in a working example that I have built called “Race Replica”:

www.racereplica.org

The code itself is object-orientated Javascript using prototypes.

Race Replica is hosted on an AWS S3 bucket, with cloud-front. The website also includes an algorithm, written in javascript, which adjusts the column height of text to give it a magazine style.
